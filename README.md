## Rulează local

**Prerequisites:**  Node.js


1. Instalează dependințe:
   `npm install`
2. Setează `GEMINI_API_KEY` în [.env.local](.env.local) la Gemini API key
3. Rulează aplicația:
   `npm run dev`
